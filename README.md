# Pine-Actions
PineActions is a basic re-implementation of some [Moto Actions](https://play.google.com/store/apps/details?id=com.motorola.actions&hl=en_US&gl=US) for the pinephone.

## Currently implemented actions:

 - [ ] Twist camera
 - [x] Twist torch
